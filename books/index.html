Priest / 2016-9 昕文化工作室 / 9.0 分
2017-01-06 / Priest / ★★★★★ 力荐
P大的小说，质量有保证。《杀破狼》是经典，称赞的话不多说，我想说点儿自己阅读的感受。一，P大的小说，剧情上更胜一筹，《杀破狼》如此，在剧情设定和推动上，远比感情戏份更加吸引人；再者，P大的文，三观根苗正红，读起来实在很容易受感染，最喜欢书中的一句话“有人心异变，三头五年就面目全非，也有人如止水，十万八千里走过，初心不改”，小说不单单有情节有情感，还有情怀；不是披着套路的小说，字里行间有作者要传达的精神。二，感情处理有些纰漏，对于长庚和顾昀的感情，来得有些快，长庚的感情很自然，但顾昀，从最初有些尴尬地抵触，到自然而然接受了长庚，这中间的过渡和转变我觉得并不明显；但小小瑕疵，的瑕不掩瑜。三，背景设定新颖，皇帝和将军，皇权军权的分立，猜忌和阴谋；蛮人和西洋人的外侵，重机甲武器装备等等，着实精彩。

2016-10-29 / 古言 鹤唳华亭 / ★★★★★ 力荐
我觉得，《鹤唳华亭》更像是诗，满腹情怀的诗篇。定权的“妇人之仁”，执意不肯牺牲身边人来上位，宁可被算计；就连最后的“逆谋”，都是远在塞外的表哥的“逼迫”。每次他不下决心，左右之人便劝谏莫有妇人之仁，就连他的对手也道出他的妇人之仁。可萧定权还是不予。定权生错了时，投错了胎，像个执拗的小孩子，贪恋着温暖又不得不背负着沉重前进。定权与阿宝，最让我难忘的情节是，最后定权成为废太子被关，阿宝来看他，萧定权说：“愿与君，生生世世，永不再晤。”
